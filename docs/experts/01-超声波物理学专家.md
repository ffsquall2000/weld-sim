# 超声波金属焊接物理学深度分析——以锂电池焊接为核心场景

> **专家角色**：超声波物理学与声学专家，30年超声波金属焊接研究经验

---

## 一、超声波振动参数

### 1.1 频率

**物理本质：** 超声波频率 f 决定了焊接系统的振动周期 T = 1/f，也直接决定了单位时间内界面经历的摩擦循环次数。从波动方程出发，超声波在固体介质中的传播满足：

```
∂²u/∂t² = c² × ∂²u/∂x²
```

其中 c 为介质中的声速，u 为质点位移。波长 λ = c/f，对于铝（c ≈ 6300 m/s），20 kHz 时波长约 315 mm，40 kHz 时约 158 mm。

**为何锂电池焊接常用 20 kHz 和 40 kHz：**

- **20 kHz** 是人耳听觉阈值的上限，这是工程上的基本下界——低于此频率会产生严重噪声问题。但更本质的原因在于：20 kHz 系统波长较长，允许更大的焊头尺寸和更大的振幅输出（通常 20-50 μm peak-to-peak），适合较厚的铜箔/铝箔多层叠焊（如锂电池极耳焊接中 40-80 层铝箔的情形）。较大振幅能提供足够的界面剪切位移，克服多层金属间的氧化膜和接触电阻。

- **40 kHz** 波长减半，焊头尺寸更紧凑，振幅通常较小（5-20 μm），但频率翻倍意味着单位时间内的摩擦循环次数加倍。这适合薄材、精密焊接场景（如极耳与集流体的单层或少层焊接），能量输入更精细可控，热影响区更小，对薄铜箔（6-12 μm）的损伤风险更低。

- **频率选择的深层物理逻辑：** 焊接功率可表达为 P ∝ f · A²（A 为振幅），这意味着高频低振幅与低频高振幅可以输入相近的总功率，但能量的时间分布特性不同。高频产生更均匀、更细密的界面塑性变形，而低频产生更剧烈的宏观滑移。对于铝这类较软的金属，20 kHz 的大振幅更有效；对于铜这类较硬的金属，有时需要 40 kHz 的高频密集冲击来实现氧化膜的有效破碎。

### 1.2 振幅

**物理本质：** 振幅 A 是焊头表面质点的最大位移量，直接决定了焊接界面的剪切应变量。焊头表面的瞬时速度为：

```
v(t) = 2πfA × cos(2πft)
```

最大速度 v_max = 2πfA，最大加速度 a_max = (2πf)²A。

以 20 kHz、振幅 30 μm 为例：v_max ≈ 3.77 m/s，a_max ≈ 4.74 × 10⁵ m/s²（约 48000 g）。这一极端加速度是理解超声波焊接界面微观行为的关键——它意味着界面金属在每个振动周期内都经历剧烈的惯性力作用。

**振幅与能量传递的关系：**

焊接界面单位面积的瞬时功率密度可近似为：

```
p(t) = σ_shear(t) · u̇(t)
```

其中 σ_shear 为界面剪切应力，u̇ 为界面相对滑移速度。对于库仑摩擦模型，σ_shear = μ · σ_n（μ 为摩擦系数，σ_n 为法向压力），则每周期耗散能量为：

```
E_cycle = 4μσ_n × A × A_contact
```

每秒输入的总摩擦功率为：

```
P_friction = 4fμσ_n × A × A_contact
```

这表明功率与频率和振幅均成线性关系（在纯库仑摩擦假设下）。但实际焊接中，随着界面温度升高、氧化膜破碎和金属软化，摩擦机制会从库仑摩擦过渡到粘着-滑移（stick-slip）再到塑性变形主导，使得振幅的影响变得非线性。

**振幅的临界效应：** 存在一个最小临界振幅 A_cr，低于此振幅时界面完全处于粘着状态，无相对滑移发生，焊接无法启动。A_cr 取决于法向载荷、材料屈服强度和界面摩擦特性。对于锂电池铝极耳焊接，典型的 A_cr 约为 5-8 μm。

### 1.3 振动模式

**纵向振动（Longitudinal Mode）：** 质点位移方向与波传播方向平行。在超声波金属焊接中，这是最常用的模式。焊头做平行于焊接界面的往复运动（线性振动），在界面产生剪切力。物理上，这种模式最有效地破碎界面氧化膜——氧化物（如 Al₂O₃，莫氏硬度 9）是脆性材料，在交变剪切应力下极易碎裂并被挤出焊接区域，暴露出新鲜金属面。

**扭转振动（Torsional Mode）：** 焊头绕其轴线做扭转往复运动，界面产生旋转剪切。优势在于焊接区域内各点的相对滑移量更均匀（线性振动模式下，平行于振动方向的不同位置滑移相位相同但垂直方向无滑移），但系统设计更复杂。在某些圆形焊点的锂电池连接场景中有应用价值。

**横向振动（Transverse/Flexural Mode）：** 质点位移垂直于波传播方向。在焊接界面产生法向的锤击效应而非剪切效应，通常不直接用于金属焊接（更多用于塑料焊接中的垂直振动模式）。但在实际系统中，由于模态耦合，焊头的主纵向振动往往伴随微量的横向分量，这反而有助于维持界面接触。

**锂电池场景的选择逻辑：** 锂电池极耳焊接绝大多数采用线性纵向振动。原因在于：(1) 极耳为薄带状结构，线性振动方向可与极耳长度方向平行，应力分布合理；(2) 多层薄箔焊接需要强烈的层间剪切来逐层破碎氧化膜并实现固态扩散连接；(3) 线性振动系统技术成熟，频率控制精度高。

### 1.4 谐波与驻波效应

**驻波的形成：** 超声波在焊头和工件中传播时，由于端面和界面的反射，会形成驻波。驻波的位移分布为：

```
u(x,t) = 2A × sin(kx) × cos(ωt)
```

其中 k = 2π/λ 为波数。驻波的节点（displacement node，位移为零、应力最大）和腹点（antinode，位移最大、应力为零）的分布直接决定了焊头的设计。焊头的安装法兰必须位于节点位置，否则振动能量会通过安装点大量泄漏。

**谐波问题：** 实际焊接系统并非完美的单频振动。由于压电陶瓷的非线性、材料阻尼的非线性以及界面摩擦的非线性，系统会产生高次谐波（2f、3f 等）。在锂电池多层焊接中，这些谐波分量可能导致：
- 某些薄箔层出现局部共振，引起应力集中和撕裂
- 焊接能量在焊头表面的不均匀分布
- 焊接噪声增加（高次谐波的一部分落入可听范围）

**焊头中的模态耦合：** 当焊头的横向尺寸接近半波长时，会出现纵向振动与弯曲振动的耦合，产生寄生模态。这在大面积焊头（锂电池大极耳焊接用）设计中是核心挑战——焊头工作面必须保持均匀的振幅分布，任何寄生模态都会导致焊点质量不一致。通过开槽（slotting）设计可以抑制寄生模态，将大面积焊头分割为若干独立的振动单元。

## 二、能量传递机制

### 2.1 超声波能量向界面热能的转化

超声波焊接的能量转化链条可归纳为：

```
电能 → [压电效应] → 机械振动能 → [声学传递] → 界面摩擦/变形能 → [不可逆过程] → 热能 + 塑性变形功
```

需要特别强调的是，超声波金属焊接**不是**一个以热为主导的连接过程。其本质是固态连接——界面温度通常仅达到母材熔点的 25%-50%（对于铝，界面温度约 150-300°C，远低于 660°C 的熔点）。真正驱动连接的是：

1. **氧化膜的机械破碎与移除**：交变剪切应力使脆性氧化层碎裂
2. **新鲜金属面的亲密接触**：在压力和超声波辅助下，无氧化膜的金属面达到原子级距离（< 1 nm）
3. **固态扩散与金属键形成**：在升温和塑性变形辅助下，界面原子发生互扩散

热能在此过程中的角色是辅助性的——它软化金属、降低屈服强度、加速扩散，但并非连接的直接驱动力。

### 2.2 界面摩擦机制

**微观滑移（Microslip）阶段：** 焊接初期，两个工件表面仅在微凸体（asperity）顶端接触，实际接触面积远小于表观面积（通常仅为 1%-5%）。超声波振动首先在这些微凸体接触点产生微观滑移，滑移区从接触边缘向中心扩展（符合 Mindlin 接触理论）：

```
δ_slip = (μP/k_t) × [1 - (1 - Q/(μP))^(2/3)]
```

其中 P 为法向力，Q 为切向力，k_t 为切向接触刚度。当 Q = μP 时，整个接触面进入全滑移状态。

**塑性变形主导阶段：** 随着界面温度升高和微凸体被压平，实际接触面积急剧增加，界面过渡到以塑性变形为主的能量耗散模式。此时界面剪切应力不再遵循库仑摩擦，而是由材料的剪切屈服强度 τ_y 决定：

```
σ_shear = τ_y(T) ≈ σ_y(T) / √3
```

其中 σ_y(T) 为温度依赖的屈服强度。对于铝，σ_y 随温度显著下降，这意味着随着焊接进行，界面的剪切阻力降低，能量耗散率先升后降——这是焊接功率-时间曲线呈现典型峰值的物理原因。

**粘着-滑移（Stick-Slip）机制：** 在微滑移和全塑性变形之间，存在一个粘着-滑移的过渡阶段。在每个振动周期内，界面交替经历粘着（静摩擦）和滑移（动摩擦），产生非正弦的界面剪切力波形。这个阶段的能量耗散效率最高，也是有效焊接窗口的核心时段。

### 2.3 声阻抗匹配

**基本原理：** 声阻抗定义为 Z = ρc（ρ 为密度，c 为声速）。当超声波从介质 1 传入介质 2 时，界面的能量透射系数为：

```
T = 4Z₁Z₂ / (Z₁ + Z₂)²
```

各材料的声阻抗：
- 钢（焊头材料）：Z ≈ 46 × 10⁶ kg/(m²·s)
- 铝：Z ≈ 17 × 10⁶ kg/(m²·s)
- 铜：Z ≈ 42 × 10⁶ kg/(m²·s)
- 钛合金（焊头材料）：Z ≈ 27 × 10⁶ kg/(m²·s)

**锂电池焊接中的阻抗匹配问题：**

对于铝极耳焊接，钢焊头到铝的透射系数约为 0.79，存在约 21% 的反射损失。使用钛合金焊头（Z ≈ 27 × 10⁶）可将透射系数提高到约 0.95，显著改善能量传递效率。这是近年来钛合金焊头在锂电池焊接中日益普及的物理原因。

对于铜极耳焊接，钢焊头到铜的阻抗匹配天然良好（T ≈ 0.997），但铜的高热导率（约 400 W/(m·K)，是铝的近两倍）意味着界面产生的热量被迅速传导走，使界面温度难以累积。这是铜焊接比铝焊接更困难的物理根源——不是阻抗匹配问题，而是热量快速耗散导致界面软化不足。

**多层叠焊的阻抗问题：** 在锂电池极耳 40-80 层薄箔叠焊中，每个层间界面都是一个声阻抗不连续点（氧化膜、空气间隙、接触不良）。超声波在穿越这些界面时发生多次反射和透射，形成复杂的干涉场。底层箔片接收到的能量可能仅为顶层的 20%-40%。这就是为什么多层焊接需要优化砧座（anvil）设计——砧座表面的齿纹结构通过"锚定"底层箔片，使其与砧座之间不发生相对滑移，从而将超声波能量集中在需要焊接的层间界面。

### 2.4 能量衰减与传递效率

超声波在金属中传播的衰减机制包括：

1. **吸收衰减：** 由材料的内摩擦（粘弹性阻尼）引起，衰减系数 α_abs ∝ fⁿ（n 取决于衰减机制，多晶金属中通常 n ≈ 1-2）
2. **散射衰减：** 由晶粒边界的散射引起，当波长远大于晶粒尺寸时（Rayleigh 散射），α_scat ∝ d³f⁴（d 为晶粒尺寸）
3. **界面衰减：** 多层结构中层间界面的反射和模式转换引起的能量损失

对于 20 kHz 频率，波长（铝中约 315 mm）远大于典型晶粒尺寸（10-100 μm），因此散射衰减几乎可以忽略。体吸收衰减也很小——20 kHz 超声波在铝中的衰减长度为数米量级。这意味着能量从焊头到焊接界面的传输损失主要来自界面反射，而非体积衰减。

**整体系统效率：** 从电能输入到有效焊接能量，典型的效率链为：
- 电源到换能器（电-机械转换）：85%-95%
- 换能器到变幅杆到焊头（声学传输）：90%-98%
- 焊头到工件界面（耦合效率）：60%-90%（取决于阻抗匹配和接触条件）
- 界面能量中有效用于焊接的比例：30%-60%（其余以热耗散损失）

总体而言，从电能输入到有效焊接能量的转换效率约为 15%-45%，锂电池焊接场景中，经过优化后通常可达 25%-35%。

## 三、声学系统变量

### 3.1 换能器（Transducer）

**压电效应基础：** 超声波焊接换能器利用逆压电效应——对压电陶瓷（通常为 PZT，即锆钛酸铅 Pb(Zr_xTi_{1-x})O₃）施加交变电场时，产生机械形变。

**关键参数：**

- **机电耦合系数 k：** 衡量电能与机械能的转换效率，定义为 k² = 转换的机械能/输入的电能。PZT-4（常用于大功率超声波焊接）的 k₃₃ ≈ 0.70，意味着约 49% 的电能可转化为纵向振动能。
- **机械品质因子 Q_m：** 衡量换能器在谐振频率附近的能量储存与耗散之比。Q_m 越高，谐振峰越尖锐，能量转换效率越高。PZT-4 的 Q_m ≈ 500，适合大功率连续工作；PZT-8 的 Q_m ≈ 1000，更适合超大功率应用但 d 系数较低。
- **居里温度 T_c：** PZT-4 的 T_c ≈ 328°C，工作温度不应超过 T_c/2 ≈ 164°C，否则压电性能会不可逆退化。这限制了换能器的最大连续功率。

**谐振频率漂移：** 换能器的谐振频率 f_r 取决于其有效长度和声速：

```
f_r = c_eff / (2L_eff)
```

温度升高会导致：(1) PZT 陶瓷的弹性模量变化（软化），声速降低；(2) 热膨胀使有效长度增加。两者都导致谐振频率下降。典型的温升频率漂移约为 -2 至 -5 Hz/°C。在锂电池高速焊接产线上，换能器连续工作温升可达 30-50°C，频率可漂移 100-250 Hz，这对频率跟踪系统提出了严格要求。

**预紧力的影响：** Langevin 型换能器（最常见的超声波焊接换能器结构）由前后金属质量块夹持 PZT 陶瓷片组成，通过中心螺栓施加预紧力。预紧力不足会导致陶瓷在拉伸半周期脱粘（PZT 陶瓷抗拉强度很低），产生非线性和谐波；预紧力过大会改变陶瓷的极化状态，降低 d 系数。最优预紧力通常为陶瓷抗压强度的 10%-25%。

### 3.2 变幅杆（Booster/Horn Amplifier）

**物理功能：** 变幅杆是一个声学阻抗变换器，通过截面积的渐变将换能器输出的低振幅-高力振动变换为高振幅-低力振动（类比于电学中的变压器）。

**增益比的推导：** 对于最简单的阶梯型变幅杆，根据动量守恒和连续条件：

```
A_out / A_in = S_in / S_out = N
```

其中 S 为截面积，N 为增益比。典型的增益比为 1.5:1 到 3:1。指数型和悬链型变幅杆可以获得更大的增益比（4:1 到 6:1），同时具有更均匀的应力分布。

**应力分布：** 变幅杆中的应力与位移梯度成正比：

```
σ(x) = E × ∂u/∂x
```

在截面突变处（阶梯型变幅杆），应力集中显著，容易产生疲劳裂纹。指数型和高斯型变幅杆通过平滑的截面过渡分散应力，但加工成本更高。在高功率锂电池焊接应用中（焊接功率 2-5 kW），变幅杆的疲劳寿命是一个关键的工程考量。

### 3.3 焊头（Sonotrode/Tool Horn）

**焊头设计的核心挑战：** 焊头是直接与工件接触的部件，其设计需要同时满足：

1. **谐振条件：** 焊头的纵向长度必须为半波长（λ/2）的整数倍
2. **振幅均匀性：** 焊头工作面的振幅分布必须均匀
3. **横向模态抑制：** 焊头的宽度和高度必须远小于横向振动的半波长
4. **耐磨性和导热性：** 在高频高压条件下与金属工件接触

**焊头材料选择：** 锂电池焊接焊头通常选用：
- **钛合金（Ti-6Al-4V）：** 声学损耗低（Q_m 高），密度适中，与铝工件的阻抗匹配最佳
- **高速钢/工具钢：** 耐磨性好，但声学损耗较高
- **硬质合金涂层：** 在钛合金基体上进行表面硬化处理，兼顾声学性能和耐磨性

### 3.4 系统谐振频率跟踪

**为什么需要频率跟踪：** 超声波焊接系统的效率在谐振频率处达到最大值。偏离谐振频率即使只有几十赫兹，功率传输效率也可能下降 50% 以上。

**频率跟踪的物理方法：**

1. **锁相环（PLL）跟踪：** 监测电压和电流的相位差。在谐振频率处，换能器的阻抗呈纯阻性，电压-电流同相。PLL 控制器调整驱动频率使相位差保持为零。典型的跟踪带宽为 ±500 Hz，响应时间 < 1 ms。

2. **阻抗分析跟踪：** 持续扫描换能器的电阻抗谱，实时定位最小阻抗点（串联谐振）或最大阻抗点（并联谐振）。

3. **自适应频率控制：** 基于焊接过程中的功率反馈，通过算法（如扰动观测法）寻找最大功率传输点。

## 四、时域与频域特征

### 4.1 焊接过程中频率的动态变化

**频率漂移的物理图像：** 超声波焊接过程中，系统谐振频率呈现特征性的动态变化轨迹，可分为三个阶段：

- **阶段 I（初始接触，0-50 ms）：** 焊头刚接触工件，界面为松散接触，等效负载刚度低。此阶段频率变化剧烈且不稳定。
- **阶段 II（稳定焊接，50-300 ms）：** 界面进入稳定的摩擦-变形状态，随着界面温度升高，材料软化，谐振频率持续下降。典型的频率下降速率为 50-200 Hz/s。
- **阶段 III（焊接完成，300+ ms）：** 界面金属键基本形成，工件从"两片松散板"变为"一个整体"，谐振频率出现回升或趋于稳定。

### 4.2 功率-时间曲线的典型特征

功率-时间曲线是超声波焊接过程最重要的实时监测信号之一：

- **初始功率跃升（Power Spike）：** 焊头接触工件瞬间的短暂尖峰
- **功率爬升段（Ramp-Up）：** 界面从微滑移过渡到全滑移
- **功率峰值（Peak Power）：** 界面摩擦功率达到最大值
- **功率下降段（Decay）：** 界面温度持续升高，金属软化，功率自然下降
- **功率稳定段（Plateau）：** 焊接界面基本形成固态连接

**锂电池焊接中的功率曲线特殊性：** 多层薄箔焊接的功率曲线可能呈现多峰特征——每个峰对应一组层间界面的依次激活和焊接。

### 4.3 阻抗变化与焊接质量的关系

换能器的电阻抗包含电气部分和机械部分，焊接过程中的阻抗变化会直接映射到换能器的电阻抗上：

- **阻抗实部（电阻分量）：** 反映系统的总能量耗散率
- **阻抗虚部（电抗分量）：** 反映系统的储能特性
- **阻抗圆图（Nyquist Plot）：** 良好焊接的阻抗轨迹呈现规律的环形路径

## 五、变量间的相互关联——系统级视角

核心的耦合关系包括：

1. **频率-振幅-功率三角：** 频率和振幅共同决定界面功率密度，而功率又通过温度升高反馈影响材料软化、摩擦系数变化，形成闭环。

2. **振幅-压力-变形耦合：** 振幅决定界面剪切位移，压力决定法向载荷和接触面积，两者共同决定界面的摩擦状态。

3. **声学系统-工件负载交互：** 焊头的几何设计决定了其固有的频率响应特性，而工件负载又动态改变了系统的有效边界条件。

4. **时间-空间-能量耦合：** 焊接时间决定总能量输入，但能量在空间和时间上的分布共同决定最终的焊接质量。

从第一性原理的角度来看，超声波金属焊接的本质是一个**受迫振动系统在非线性负载条件下的能量传递与转化过程**。焊接质量最终取决于在正确的时间窗口内，向正确的空间位置（焊接界面），以正确的速率输入正确总量的机械能。
